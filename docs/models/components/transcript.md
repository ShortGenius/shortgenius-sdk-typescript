# Transcript

Transcript of the audio with timestamps. Only available after the audio is generated.

## Example Usage

```typescript
import { Transcript } from "shortgenius/models/components";

let value: Transcript = {
  words: [
    {
      text: "<value>",
      start: 5866.19,
      end: 2385.9,
    },
  ],
};
```
