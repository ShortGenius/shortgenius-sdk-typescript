# Series
(*series*)

## Overview

Create a series of videos on any topic.

### Available Operations

* [postSeries](#postseries) - Create series
* [getSeries](#getseries) - List series
* [getSeriesId](#getseriesid) - Get series

## postSeries

### Example Usage

```typescript

const shortGenius = new ShortGenius({
  bearerAuth: process.env["SHORTGENIUS_BEARER_AUTH"] ?? "",
});

  // Handle the result

run();
```
